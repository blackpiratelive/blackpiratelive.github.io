Today I am launching a new service for my fellow indieweb webmasters. I wanted to make a all-in-one stop for all the dynamic features that you might need if you are are running a static web site. 
**You can sign up today at https://tools.blackpiratex.com/**

### Privacy
I do not require any email to sign up. You can have your own username and a password to sign up. If you want to receive notifications or be able to reset password I strongly encourage you to set up telegram chat id in the settings. 

### Features:
I have made this to be most feature complete so you do not have to sign up for multiple sites for different features. 
1. **Guestbook**: You can have your own guestbook at /u/username. I also provide api for you to use that in your own site with js, so your guestbook lives inside your website and not on some random doamin. I also offer embedding, but it is recommended to use the api to submit the guestbook entries. You can learn more about it and how to use the api, here: https://tools.blackpiratex.com/docs/guestbook

2. **Comments**: I know it is not very popular idea to have comments for your blog, as it get mostly filled with spam. But for some reason if you want it on your site, you can add it. I provide api for it as well. You do not need to embed anything. If you have more than one blog, you can set up different section for comments, so they stay seperate. Learn more here: https://tools.blackpiratex.com/docs/comments

3. **Contact Forms**:  Having a contact form is a must nowadays. Yes even if you are a indiweb webmaster. I have gotten so many messages and made friends who initially contacted me through the form. I know you can use emails for that but come on..? You can have different fields for contact forms such as text, email, radio buttons, dropdown menu, url, phone, number, checkbox, textarea and much more. Learn more here: https://tools.blackpiratex.com/docs/contact-forms

4. **Likes**: Are you someone who is using hugo jelyll or any of those static site builders? Do you want to have a like or toast button just like bearblog does? Read the docs here: https://tools.blackpiratex.com/docs/likes
